* What’s a Web Stack :
	A web stack is a collection of web development software that includes, at a minimum, an operating system (OS), a programming language, database software, and a web server. A solution stack is a web stack, also known as a web application stack.
* MERN : 
	MERN is an acronym that stands for MongoDB, Express, React, and Node, the four key technologies that comprise the stack. MongoDB is a database for documents. Node.js web framework Express(.js). React(.js) is a JavaScript framework for the client side.
* Benefits
	The MERN stack is an excellent choice for companies wishing to develop high quality web applications. Indeed, this stack, in addition to using high-performance and **customized technologies**, allows for **web applications and software to be developed very quickly!**
