# Configuration

The configuration of a Strapi project lives in the `/config` folder:

```
. # root of the project
├──── config
│     ├ api.js|ts
│     ├ admin.js|ts
│     ├ cron-tasks.js|ts
│     ├ database.js|ts
│     ├ middlewares.js|ts
│     ├ plugins.js|ts
│     └ server.js|ts
```

_The block above is an excerpt from the project structure. You can click on any file name to read the corresponding documentation.

## Base configurations

From the `/config` folder, you can find and define the following base configurations:

| Configuration topic                                                                                                      | File path          | Required or optional                                                                            |
| ------------------------------------------------------------------------------------------------------------------------ | ------------------ | ----------------------------------------------------------------------------------------------- |
| [https://docs.strapi.io/dev-docs/configurations/database](https://docs.strapi.io/dev-docs/configurations/database)       | config/database    | Required                                                                                        |
| [https://docs.strapi.io/dev-docs/configurations/server](https://docs.strapi.io/dev-docs/configurations/server)           | config/server      | Required                                                                                        |
| [https://docs.strapi.io/dev-docs/configurations/admin-panel](https://docs.strapi.io/dev-docs/configurations/admin-panel) | config/admin       | Required                                                                                        |
| [https://docs.strapi.io/dev-docs/configurations/middlewares](https://docs.strapi.io/dev-docs/configurations/middlewares) | config/middlewares | Required                                                                                        |
| [https://docs.strapi.io/dev-docs/configurations/api](https://docs.strapi.io/dev-docs/configurations/api)                 | config/api         | Optional, used to define some general settings for responses and other REST-related parameters. |
